6.HTML Attribute

        

            

            Attribute
            Description

            

                alt
            An alternative text for an image,when the image cannot displayed

            

            disabled
            An input element should be disabled

            

            href
            The URL(web address) for a link

            

            id
            A unique id for an element

            

            src
            The URL for an image

            

            style
            An inline CSS style for an element

            

            title
            Extra information about an element(displayed as a tool tip)
